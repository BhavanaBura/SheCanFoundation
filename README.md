🌸 She Can Foundation Website

A simple and responsive website created for the She Can Foundation Web Development Internship Task.

The website aims to spread awareness about women's empowerment and encourage people to participate as volunteers.



🚀 Features

 ✅ Home Section
- Hero section with a motivational message
- Call-to-action Volunteer button

 ✅ About Section
- Information about the foundation
- Women's empowerment image

 ✅ Impact Section
- Education initiatives
- Skill development programs
- Community support activities

✅ Join Us Section
- Volunteer recruitment section
- Interactive button

 ✅ Responsive Design
- Works on desktops, tablets, and mobile devices

✅ Animations
- Smooth fade-in animation
- Hover effects on buttons and cards

 ✅ Modern UI
- Gradient backgrounds
- Clean layout
- Attractive color scheme

 ⚙️ How to Run the Project

Method 1: Open Directly

1. Download the project files.
2. Open `index.html` in your browser.

 Method 2: VS Code Live Server

1. Open the project folder in VS Code.
2. Install the Live Server extension.
3. Right-click on `index.html`.
4. Click **Open with Live Server**.


 🌐 Deployment

The website can be deployed using:

- GitHub Pages
- Netlify
- Vercel

GitHub Pages

1. Push the project to GitHub.
2. Go to Repository Settings.
3. Open Pages.
4. Select the `main` branch.
5. Save and deploy.
